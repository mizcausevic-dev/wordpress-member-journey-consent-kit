# Security policy

## Supported versions

Only the latest tagged release is supported for security fixes.

## Reporting a vulnerability

Please open a private security advisory or contact through `kineticgain.com/contact`.

## Scope notes

- All data in this repo is synthetic demonstration data.
- This repo does not process live regulated submissions.
- Production ingestion systems handling sensitive or regulated content require formal review, access control, retention controls, and deployment hardening before operational use.
